# VS Code Shortcuts for Windows

## Common Editor Shortcuts
- `Ctrl + P` : Quick open file
- `Ctrl + Shift + P` : Open Command Palette
- `Ctrl + ,` : Open Settings
- `Ctrl + B` : Toggle sidebar visibility
- `Ctrl + \` : Split editor
- `Ctrl + 1 / 2 / 3` : Focus editor group
- `Ctrl + Tab` : Switch between open editors
- `Ctrl + Shift + N` : New window
- `Ctrl + Shift + W` : Close window
- `Ctrl + S` : Save file
- `Ctrl + Shift + S` : Save all files
- `Ctrl + W` : Close editor

## Navigation
- `Ctrl + G` : Go to line
- `Ctrl + F` : Find
- `Ctrl + H` : Replace
- `F3` / `Shift + F3` : Find next / previous
- `Ctrl + D` : Select next occurrence
- `Ctrl + L` : Select current line
- `Ctrl + Shift + O` : Go to symbol
- `F12` : Go to definition
- `Alt + F12` : Peek definition
- `Ctrl + K F12` : Open peek references
- `Ctrl + -` : Navigate back
- `Ctrl + Shift + -` : Navigate forward

## Editing
- `Ctrl + /` : Toggle line comment
- `Shift + Alt + A` : Toggle block comment
- `Ctrl + K, Ctrl + C` : Add line comment
- `Ctrl + K, Ctrl + U` : Remove line comment
- `Alt + Up / Alt + Down` : Move line up/down
- `Shift + Alt + Up / Shift + Alt + Down` : Copy line up/down
- `Ctrl + Shift + K` : Delete line
- `Ctrl + Enter` : Insert line below
- `Ctrl + Shift + Enter` : Insert line above
- `Ctrl + Space` : Trigger suggestion
- `Ctrl + Shift + Space` : Trigger parameter hints
- `Ctrl + Shift + F` : Search in files

## Terminal
- `Ctrl + `` : Toggle integrated terminal
- `Ctrl + Shift + `` : Create new terminal
- `Ctrl + C` : Copy selected terminal text
- `Ctrl + V` : Paste into terminal
- `Ctrl + Shift + C` : Copy path of active file
- `Ctrl + Shift + \\` : Split terminal
- `Ctrl + `[` : Focus previous terminal
- `Ctrl + `]` : Focus next terminal

## Window and Layout
- `Ctrl +\` : Split editor
- `Ctrl + K Z` : Toggle Zen Mode
- `Ctrl + K M` : Change language mode
- `Ctrl + J` : Toggle panel
- `Ctrl + Shift + E` : Show Explorer
- `Ctrl + Shift + F` : Show Search
- `Ctrl + Shift + G` : Show Source Control
- `Ctrl + Shift + D` : Show Run and Debug
- `Ctrl + Shift + X` : Show Extensions

## Useful General Shortcuts
- `Ctrl + A` : Select all
- `Ctrl + Z` : Undo
- `Ctrl + Y` : Redo
- `Ctrl + F4` : Close editor
- `Alt + F4` : Close window
- `Ctrl + K Ctrl + S` : Keyboard shortcuts reference
- `F1` : Show Command Palette

---

> Note: Some shortcut combinations may vary if you have custom keybindings or extensions installed.
