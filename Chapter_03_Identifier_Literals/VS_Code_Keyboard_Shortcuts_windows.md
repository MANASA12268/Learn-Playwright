# VS Code Keyboard Shortcuts (Windows) - Table View

| Category | Shortcut | Description |
|----------|----------|-------------|
| **File Operations** | `Ctrl+S` | Save current file |
| | `Ctrl+Shift+S` | Save all files |
| | `Ctrl+O` | Open file |
| | `Ctrl+N` | New file |
| | `Ctrl+Shift+N` | New window |
| **Editing** | `Ctrl+Z` | Undo |
| | `Ctrl+Y` | Redo |
| | `Ctrl+X` | Cut |
| | `Ctrl+C` | Copy |
| | `Ctrl+V` | Paste |
| | `Ctrl+A` | Select all |
| | `Ctrl+D` | Select next occurrence (multi-cursor) |
| | `Ctrl+Shift+L` | Select all occurrences of current selection |
| | `Alt+Shift+Arrow` | Column selection (box selection) |
| **Navigation** | `Ctrl+P` | Quick open (go to file) |
| | `Ctrl+Shift+P` | Command palette (show all commands) |
| | `Ctrl+G` | Go to line |
| | `Ctrl+Tab` | Switch between open files |
| | `Ctrl+Shift+Tab` | Switch between open files (reverse) |
| | `Ctrl+Shift+O` | Go to symbol in file |
| | `F12` | Go to definition |
| | `Alt+F12` | Peek definition |
| **Search & Replace** | `Ctrl+F` | Find in current file |
| | `Ctrl+H` | Replace in current file |
| | `Ctrl+Shift+F` | Find in workspace |
| | `Ctrl+Shift+H` | Replace in workspace |
| **Code Editing** | `Ctrl+/` | Toggle line comment |
| | `Ctrl+Shift+/` | Toggle block comment |
| | `Alt+Shift+F` | Format document |
| | `Ctrl+K Ctrl+F` | Format selection |
| | `Ctrl+Shift+I` | Format document (alternative) |
| | `F2` | Rename symbol |
| **Debugging** | `F5` | Start debugging |
| | `F10` | Step over |
| | `F11` | Step into |
| | `Shift+F11` | Step out |
| | `Shift+F5` | Stop debugging |
| **Terminal** | `Ctrl+\`` | Toggle integrated terminal |
| | `Ctrl+Shift+\`` | Create new terminal |
| **Window Management** | `Ctrl+B` | Toggle sidebar |
| | `Ctrl+Shift+E` | Focus on explorer |
| | `Ctrl+Shift+G` | Focus on source control |
| | `Ctrl+Shift+D` | Focus on debug |
| | `Ctrl+Shift+X` | Focus on extensions |
| **Split Editor** | `Ctrl+\` | Split editor |
| | `Ctrl+1/2/3` | Focus on editor group 1/2/3 |
| | `Ctrl+Shift+PgUp/PgDn` | Move editor left/right |

**Tip:** You can customize these shortcuts by going to File → Preferences → Keyboard Shortcuts, or press `Ctrl+K Ctrl+S`.
